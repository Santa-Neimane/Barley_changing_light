# This project summarises all the statistical tests and visualisations used to prepare "Plant response to UV radiation in diffuse sunlight".
# The structure and content deviates from conventional code writing principles of avoiding repetability to allow the reader easily follow each individual part.

# The experiment was carried out in Helsinki, Finland in 2022 summer.
# We tested the effects of diffuse radiation (filter type, clear/diffuse) and spectral treatment (Full spectrum, UV-A long, No UV).
# Detailed description of the study please find here: "Plant response to UV radiation in diffuse sunlight" at Agricultural and Forest Meteorology.
# Data can be found in Zenodo, https://zenodo.org/records/18600056
